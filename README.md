# Social Up - Performance de Mídia Social (SQL + Power BI)

Este projeto simula a análise de desempenho de campanhas em diferentes plataformas sociais, com base em dados fictícios de uma agência de publicidade.

## Objetivo
Analisar campanhas digitais e criar um dashboard interativo no Power BI com os principais indicadores de performance (CTR, CPC, CPA, ROI e Engajamento).

## Estrutura das Tabelas
- **clientes**: informações sobre cada cliente da agência  
- **campanhas**: detalhes sobre as campanhas executadas  
- **metricas**: dados de alcance, engajamento, cliques e investimento  

## Consultas SQL
1. Totais por plataforma  
2. Taxas de engajamento e custo por clique  
3. Custo por aquisição (CPA) e top 3 campanhas com melhor resultado  
4. Média de investimento por cliente e plataforma  

## Dashboard no Power BI
- **Visão geral** com métricas principais  
- **Comparativo por plataforma**  
- **Tabela de desempenho detalhada**  
- **Filtros interativos** (cliente e plataforma)

## Insights
- YouTube apresenta o **maior alcance e volume de conversões**, mas com CPA mais alto.  
- TikTok entrega **melhor taxa de engajamento** com menor investimento médio.  
- Lumina Beauty e Conecta Telecom têm campanhas mais equilibradas entre engajamento e custo.  

## Tecnologias Utilizadas
- **MySQL** → modelagem e consultas SQL 
- **Power BI** → criação do dashboard   
- **GitHub** → versionamento e portfólio
